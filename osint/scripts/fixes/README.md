# Data Fix Scripts

One-off scripts for correcting historical data issues. Each script is idempotent and safe to re-run.

## Naming convention

`YYYY-MM-DD_description.ts` — date of the fix, not the date of the bug.

## Running

```bash
npx tsx scripts/fixes/2026-04-02_fix-gdelt-column-mapping.ts
```
