This provider uses two NASA GIBS layers:

1. **VIIRS_SNPP_GapFilled_BRDF_Corrected_DayNightBand_Radiance** — daily nighttime radiance tiles
2. **OSM_Land_Water_Map** — land/water classification (see note below)
